# Task App

Task App created with reactJS libarary, in this application used:
- App component & JSX 
- Expressions in JSX 
- Props
- PropTypes
- Styling
- Events
- Create a list with .map()
- State & useState Hook
- Global state
- Icons with react-icons
- JSON Server
- useEffect Hook & Fetch
- Routing

## Utils Commands

### Create new app:
`npx create-react-app app-name`
### Change directory:
`cd app-name`
### Runs the app in the development mode
 `npm start`
